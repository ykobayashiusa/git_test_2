inline

Computes the surface-curl of the shape-functions on a triangle.

Parameters
tThe triangle on which the shape-functions are defined.
posThe position at which the functions should be evaluated.

This function computes the curl of the shape-functions of order aorder on the given triangle. It is defined by

\[ \mbox{rot}(N(\mathbf{x})\mathbf{n}(\mathbf{x})) = \mbox{rot}_\Gamma\bigl(N(\mathbf{x})\bigr) = \mathbf{J}(\mathbf{J}^{\top}\mathbf{J} )^{-1} N(\mathbf{x}) \times \mathbf{n}, \]

where \(\mathbf{J}\) denotes the Jacobian of the mapping \(\chi\) from the reference triangle to space. As it depends on the particular triangle, it cannot be defined in the file shape_functions.h.
