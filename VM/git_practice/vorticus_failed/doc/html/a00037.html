Detailed Description

A three-dimensional vector in Euclidian space.

Objects of this class represent a vector in the three-dimensional Euclidean space. They can also thought of vectors \(\vec{v}\in\mathbb{R}^3\). The coordinates are stored in Cartesian format, but methods for conversion to spherical coordinates also exist. This class is called point in order to emphasize its geometric nature. For general tuples of numbers, the math::vector class should be used.
