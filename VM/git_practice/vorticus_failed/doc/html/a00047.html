Detailed Description

Planar triangle in three-dimensional space.

Objects of this type represent a planar triangle in the three-dimensional space. A triangle is always part of a geometry::mesh and is uniquely defined by its three corner nodes.

Constructor & Destructor Documentation

geometry::triangle::triangle ( )
inline

Creates an empty triangle with id 0. Only use this for array construction. Always assign the nodes afterwards.

This constructor sets the pointers to the three corner nodes to zero. The purpose of this constructor is to be able to create an array of triangles. Do not use any of the member functions before assigning the corresponding nodes!

Return normal vector of unit length.

This function returns the normal vector of unit length of the triangle. Think of the triangle nodes numbered as follows:

2 |\ | \ | \ | \ 0-—1

Then the normal is pointing towards you, out of the screen.
