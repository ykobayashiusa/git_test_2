Detailed Description

Surface meshes consisting of triangles.

Objects of this class represent meshes of surfaces in the three-dimensional space. Such a mesh consists of a number of nodes and triangles. Currently, the constructor can only be used to create a mesh which approximates a sphere.

Constructor & Destructor Documentation

geometry::mesh::mesh ( int  fineness = 0)

Creates an approximation of a sphere.

Parameters
finenessThe number of uniform refinements.

This constructor creates an icosahedron around the origin with all nodes lying on the unit sphere. Afterwards uniform refinements are applied. In a uniform refinement each triangle is split into four sub-triangles, the newly created nodes are scaled such that they lie on the unit sphere. The surface normals are pointing outside the sphere.

void geometry::mesh::uniform_refine ( )

Performs a uniform mesh-refinement.

This function splits each triangle into four equally sized sub-triangles. In order to approximate a sphere, the newly created nodes are scaled such that they lie on the unit sphere.
